API Documentation

Metric/Imperial Converter converts between select metric and imperial units.

GET a converted unit:

Use the endpoint: "https://fcc-metimp-convert-kl.glitch.me/api/convert?[query]"

Request query must have property name "input" and contain a valid number and unit.

Ex: "https://fcc-metimp-convert-kl.glitch.me/api/convert?input=3gal"

Accepted units: gal, L, mi, km, lbs, kg.

If the post is succesful, a JSON object will be returned with properties initNum, initUnit, returnNum, returnUnit and string.

Ex: { "initNum": 3,
"initUnit": "gal",
"returnNum": 11.35623,
"returnUnit": "L",
"string": "3 gallons converts to 11.35623 liters" }
